## 平台:

## 效果：
- 可更新到BIG SUR 11.6 

## MACOS FILE ADDRESS

## 如何使用

### 1. 使用 TRANSMAC 将镜像烧录到U盘中。
### 2. 打开烧录好的U盘，找到EFI文件夹，将里面的内容清理完。
### 3. 将本项目的EFI中的所有文件，全部放入。
